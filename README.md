# Ejercicios de Python · Prácticas guiadas

Repositorio con ejercicios de Python realizados como práctica guiada, organizados por bloques temáticos (variables, bucles, listas, funciones, etc.). El objetivo es consolidar bases sólidas de programación en Python orientadas a datos y desarrollo de aplicaciones.

## Contenido del repositorio

- `bloque1_basicos/`  
  Ejercicios de entrada/salida, variables, tipos básicos y condicionales (`if`, `elif`, `else`).

- `bloque2_bucles/`  
  Prácticas con bucles `for` y `while`, uso de `range()`, juegos sencillos como “adivina el número”.

- `bloque3_listas_diccionarios/`  
  Manipulación de listas y colecciones: acumuladores, búsqueda de máximos y mínimos, filtrado de elementos, cálculo de medias, etc.

- `funciones_utilidades/`  
  Módulo con funciones reutilizables (`pedir_numeros`, `max_min`, `nota_media`) y un `main.py` que actúa como punto de entrada del pequeño programa.
